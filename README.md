cheng95-stat
==========

基础接口监控系统。提供客户端接口支持！


客户端接口
==========

客户端提供的接口位于sClient目录通过执行脚本api.php发送请求


用法
=======
	
	//1.启动服务端
	php start.php start -d
	
	//2.通过ip访问服务端web
	http://xxx.xxx.xxx.xxx:40002

	//3.执行客户端请求
	客户端api.php中注意修改目标服务器ip地址（$serverLists）

开发环境地址
=======

http://192.168.1.110:40002
